# JoSAA Portal

A JEE Advanced counselling analytics portal for aspirants navigating the JoSAA seat allocation process.

## Features

- **Rank Trend Analysis** — Visualizes opening and closing ranks across years for all IITs, NITs, and IIITs
- **Smart Filters** — Filter by category (General, OBC, SC, ST), gender, and college
- **Branch Predictions** — Predicts branch availability based on your rank and preferences
- **Historical Data** — Multi-year dataset for informed decision-making

## Tech Stack

- **Frontend:** React.js
- **Backend:** Node.js, Express.js
- **Data:** JoSAA official dataset

## Getting Started

```bash
git clone https://github.com/iamadarsh05/josaa-portal
cd josaa-portal
npm install
npm run dev
```
